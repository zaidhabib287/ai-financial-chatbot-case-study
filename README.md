# AI-Powered Financial Chatbot System

A secure, rule-driven chatbot system for fund transfers and beneficiary management with RAG-based document ingestion.

## Features
- Customer chatbot for fund transfers and beneficiary management
- Admin portal for document management and compliance
- RAG pipeline for compliance document processing
- Mock APIs for financial operations
- CI/CD and MLOps pipelines

## Tech Stack
- Backend: FastAPI
- Frontend: Streamlit (Admin), React (Chatbot UI)
- LLM: Langchain + OpenAI
- Vector DB: FAISS
- Database: PostgreSQL
- Authentication: JWT
